大模型API调用方法

Author: Henry Li   Date: 2025-08-25   Category: 技术速记    本文阅读量: 次

1.Python调用

  1. 最快跑通：直接看「OpenAI 官方库方式」

  2. 想灵活切换多家模型：用「LangChain 封装方式」

  3. 想上线生产：加上「流式/异步方式」，且使用以下建议

    • 密钥：用环境变量或 .env 文件，切勿硬编码。

    • 并发：生产环境使用连接池（httpx.AsyncClient）或 LangServe，提高吞吐量。

    • 调试：打开 client = OpenAI(..., http_client=httpx.Client(verify=False, timeout=30)) 可快速定位 SSL/超时问题。

2. Java调用

  • 想 5 分钟跑通 → 用「阿里云百炼 SDK」。
  • 想零依赖对接所有 OpenAI-格式接口 → 用「HTTP 原生调用」。
  • Spring Boot 工程 → 直接上「LangChain4j」。
  1. 安全：永远不要把 key 写死在代码或 Git 里，使用环境变量或 Spring Cloud Config。
  2. 版本：dashscope-sdk-java ≥ 2.19、langchain4j ≥ 0.35，可避免 90 % 协议不兼容问题。
  3. 流式：官方 SDK 已支持 Generation.streamCall(...)；HTTP 方式把 "stream":true 打开即可。
  4. 重试 / 并发：生产环境建议用 Resilience4j 或 Sentinel 做重试、熔断与限流。
