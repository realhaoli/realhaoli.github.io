1.2 Ollama

Ollama 是一个开源、轻量级的本地大型语言模型（LLM）部署与管理工具，旨在帮助用户在本地设备（如个人电脑、服务器）上快速、便捷地运行、管理和定制各类开源大模型，无需依赖云端服务

可以理解为Ollama是大模型界的Docker

命令 说明
ollama pull llama3 下载模型
ollama run llama3 运行模型
ollama list 查看已下载模型
ollama rm llama3 删除模型
ollama cp llama3 mymodel 复制模型并自定义名称

2.5 本地知识库

本地部署的大模型完全支持接入知识库，实现“知识增强问答”（RAG，Retrieval-Augmented Generation）。
可以把本地文档、PDF、网页等资料变成向量，存进向量数据库，再通过大模型调用这些资料回答问题，数据完全留在本地，安全可控。

2.6 本地Function Calling

Ollama从 v0.3.0 起，支持通过 /api/chat 接口使用 tools 字段声明可调用的函数

通过 JSON Schema 定义函数，模型返回结构化调用指令，可在本地执行

项目 说明
模型要求 并非所有本地模型都支持 function calling，推荐使用 llama3.1 或 qwen2 系列。
显存要求 即使 int4 量化版 llama3.1:8b 也需要 4~5GB 显存 。
准确率 小模型未经微调时，function calling 准确率可能较低，建议配合 prompt 模板优化。
替代方案 如果你发现 Ollama 的 function calling 不稳定，可尝试 LM Studio 或 Open WebUI，它们提供了更完整的工具调用封装 。
