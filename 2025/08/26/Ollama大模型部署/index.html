1.2 Ollama

Ollama 是一个开源、轻量级的本地大型语言模型（LLM）部署与管理工具，旨在帮助用户在本地设备（如个人电脑、服务器）上快速、便捷地运行、管理和定制各类开源大模型，无需依赖云端服务

可以理解为Ollama是大模型界的Docker

命令 说明
ollama pull llama3 下载模型
ollama run llama3 运行模型
ollama list 查看已下载模型
ollama rm llama3 删除模型
ollama cp llama3 mymodel 复制模型并自定义名称
